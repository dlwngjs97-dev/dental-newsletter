# 🦷 치과 뉴스레터

치과 관련 주요 언론사의 뉴스를 매일 자동으로 수집하여 보여주는 웹사이트

## 📰 수집 언론사

- **치의신보** (dailydental.co.kr)
- **치과신문** (dentalnews.or.kr)
- **덴탈아리랑** (dentalarirang.com)

## ✨ 주요 기능

- 📱 반응형 디자인 (모바일/태블릿/PC)
- 🔍 언론사별 필터링
- 🖼️ 썸네일 이미지 표시
- 📄 상세 내용 보기
- 🔗 원문 링크 제공
- 🤖 매일 자동 크롤링 (오전 10시)

## 🚀 빠른 배포

```bash
./quick-deploy.sh
```

자세한 가이드: [DEPLOYMENT_GUIDE.md](./DEPLOYMENT_GUIDE.md)

## 🌐 공유 URL

배포 완료 후:
```
https://dental-newsletter-[고유코드].vercel.app
```

## 🛠️ 기술 스택

- **Frontend:** Next.js 14, TypeScript, Tailwind CSS
- **Backend:** Python, BeautifulSoup4, SQLite
- **Deploy:** Vercel + GitHub Actions
